# Vision

PDTBar is a quiet macOS menu bar companion for a Portfolio Dividend Tracker portfolio. It should surface only the few portfolio pressures worth attention now, stay local-first and read-only by default, and keep financial data descriptive rather than advisory.

## Merge by Default

- Bug fixes with clear cause in Claude login, PDT MCP readiness, data fetching, pulse rendering, or status icons.
- Small facet, pressure, attention-item, fixture, and docs improvements aligned with v1 scope.
- UI polish that preserves quiet-by-default behavior and layered disclosure.
- Smoke checks, packaging proof, and release workflow fixes.
- Reuse cleanup that respects CodexBar, RepoBar, and mcporter boundaries.

## Needs Sign-Off

- Financial advice, trade, fund movement, or PDT write behavior.
- New external data sources, LLM data sharing, or privacy-impacting behavior.
- Broad information architecture, threshold, history-store, or product-positioning changes.
- In-app updater, distribution, signing, notarization, or Homebrew policy changes.
- Runtime dependencies copied from research tools without explicit product review.
